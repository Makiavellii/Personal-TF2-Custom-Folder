# Personal TF2 Custom Folder
Github repo to house my personal TF2 custom folder.  

HUD is KBN HUD.

# Launch Options
-dxlevel 80 +clientport 27006-27014 -32bit -full -width 1920 -height 1080 -freq 144 -console -novid -nojoy -reuse -nosteamcontroller -nohltv -particles 1 -precachefontchars -r_emulate_gl  

# Binds
**E** - Move Forward  
**D** - Move Backwards  
**S** - Strafe Left  
**F** - Strafe Right  
**Q** - Weapon Slot 1  
**W** - Weapon Slot 2  
**R** - Weapon Slot 3  
**A** - Call Medic  
**SPACE** - Crouch Jump  
**MOUSE1** - Shoot  
**MOUSE2** - Scope In  
**MOUSE4** - Crouch  
**SCROLLUP** - Regular Jump
