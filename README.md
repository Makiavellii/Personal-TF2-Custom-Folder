# Personal TF2 Custom Folder
Github repo to house my personal TF2 custom folder.  

HUD is a stripped down version of broesel HUD which has a bunch of things removed which cause lag/stutters.

# Binds
**E** - Move Forward  
**D** - Move Backwards  
**S** - Strafe Left  
**F** - Strafe Right  
**Q** - Weapon Slot 1  
**W** - Weapon Slot 2  
**R** - Weapon Slot 3  
**A** - Call Medic  
**SPACE** - Scope In  
**MOUSE1** - Shoot  
**MOUSE2** - Crouch Jump  
**MOUSE4** - Crouch  
**SCROLLUP** - Regular Jump  

# Settings
**MOUSE** - Glorious Model O-  
**MOUSEPAD** - CMSTORM Swift-RX XL  
**DPI** - 1,600  
**POLLING RATE** - 1000hz  
**WINDOWS SENSITIVITY** - 6/11  
**IN GAME SENSITIVITY** - 0.57   
**EDPI** - 456 (dpi * ingame)  
**DISTANCE** - 2.9feet/360 | 35.8in/360 | 91.14cm/360  
**ZOOM SENSITIVITY RATIO** - 1  

I use the [https://github.com/a1xd/rawaccel](raw accel) driver to make 1,600 DPI feel like 800. That way I get the benefits of 1,600 DPI with the same feeling as 800 DPI.
