# Personal-TF2-Config
Github repo to house my personal TF2 Config. Download at your own risk. I can't help you if it breaks your game.

# Binds
**E** - Move Forward  
**D** - Move Backwards  
**S** - Strafe Left  
**F** - Strafe Right  
**Q** - Weapon Slot 1  
**W** - Weapon Slot 2  
**R** - Weapon Slot 3  
**A** - Call Medic  
**SPACE** - Scope In  
**MOUSE1** - Shoot  
**MOUSE2** - Crouch Jump  
**MOUSE4** - Crouch  
**SCROLLUP** - Regular Jump  
**SCROLLDWN** - Regular Jump
